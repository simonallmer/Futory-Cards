# Futory Cards

An intergalactic deckbuilding game web application.

## Running the Project

Since this project uses Vanilla HTML/CSS/JS, you can serve it using any static file server.

### Using Python (Recommended)
Run the following command in the project directory:
```bash
python3 -m http.server 8000
```
Then open [http://localhost:8000](http://localhost:8000) in your browser.

## Features
- **Intergalactic Theme**: Animated starfield background and neon aesthetics.
- **Glassmorphism UI**: Modern, translucent menu design.
- **Responsive Layout**: Works on different screen sizes.
- **Current State**: Home screen with disabled menu options and credits.
